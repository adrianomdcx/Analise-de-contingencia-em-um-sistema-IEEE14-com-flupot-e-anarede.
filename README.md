# Analise-de-contingencia-em-um-sistema-IEEE14-com-flupot-e-anarede.
Esse repositório contém códigos para inscrição de .fpo, linguagem lida pelo flupot que otimiza a análise e retirada de violações em um sistema IEEE 14 barras, os códigos acompanham o arquivo com situação normal, após a correção do software.
